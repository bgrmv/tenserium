Прототип интерфейса Tenserium (Web-версия)
1. Главный экран (Dashboard / Выбор режима)
Этот экран встречает пользователя после входа. Он задает тон всему приложению: темная тема, минимализм, фокус на действии.

Описание визуальных элементов:

Верхняя панель: Логотип Tenserium (стилизованная буква 'T' в виде молнии/секундомера), Навигация (Тренажер, Обучение, Статистика), Профиль пользователя (Аватар, Ранг "Scholar", Streak: 15 дней).

Центральный блок: Два больших интерактивных блока для выбора режима:

Normal Mode (Обычный): Описание («Все 12 времен открыты. Играй в своем темпе»). Кнопка «Начать» и иконка шестеренки для быстрой настройки пула времен.

Rank Mode (Рейтинговый): Описание («Соревнование на скорость. Открывай времена, зарабатывай очки»). Визуализация прогресса: иконка текущего ранга (бронзовый щит "Rookie"), полоса прогресса до следующего уровня и кнопка «Squad Battle (4 игрока)».

Нижняя часть: Блок «Daily Challenge» (Ежедневное задание) с таймером обратного отсчета и кнопкой «Принять вызов».

2. Игровой интерфейс (Squad Battle / Rank Mode)
Самый динамичный экран. Это момент игры, где важна каждая миллисекунда.

Описание визуальных элементов:

Центр: Огромный, легко читаемый промоутер (вопрос). В данном примере это контекстное описание на русском: «Действие, завершённое до другого события в прошлом». Чуть ниже — таймер обратного отсчета для этого вопроса (уменьшающаяся полоса).

Нижняя панель (Сетка ответов): Сетка 3x4, представляющая 12 времен. Каждый блок времени имеет свой уникальный акцентный цвет (акценты на базе цветовой кодировки из ТЗ).

Каждый блок содержит: Название времени (например, "Past Perfect"), Горячую клавишу (F7), и цветовую метку.

В Rank Mode доступные времена подсвечены, заблокированные — слегка затенены с иконкой замка.

Верхний правый угол (Счет и Комбо): Текущие очки за сессию, визуализированный индикатор комбо (например, Combos: 7✓).

Верхний левый угол (HUD соревнований): Миникарта с 4 аватарками игроков (вы и 3 соперника/бота). Рядом с каждым аватаром — полоса прогресса (сколько вопросов пройдено) и текущий счет. Это создает ощущение живого соревнования.

3. Раздел Обучения (Learn Section / Present Simple)
Этот экран показывает, как выглядит база знаний. Он чистый, структурированный и предназначен для изучения, а не для скорости.

Описание визуальных элементов:

Сетка времен (Слева): Вертикальный список всех 12 времен с их цветовой кодировкой. "Present Simple" выделен как активный.

Основной контент (Справа): Заголовок "Present Simple" (в цвете времени).

Grammar (Грамматика): Простые формулы (V / V-s).

Usage (Употребление): Пункты с иконками (Факты, Привычки, Расписания).

Examples (Примеры): Список предложений. Важно: глагольные конструкции выделены тем же акцентным цветом времени. «She WORKS every day.»

Time Markers (Маркеры времени): Список слов-маркеров (always, usually, etc.).

Кнопка действий: Большая кнопка «Тренировать Present Simple» (переход в Normal Mode с преднастроенным фильтром).

Этот прототип демонстрирует, как сухие требования ТЗ превращаются в визуально привлекательный и функциональный продукт.
